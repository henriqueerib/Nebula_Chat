# 🌌 Nebula Chat AI
> **Secure Quantum Messaging** - Uma plataforma de comunicação em tempo real com design futurista (Glassmorphism) e recursos premium.

## ✨ Funcionalidades
* **Comunicação em Tempo Real:** Powered by WebSockets (Socket.IO).
* **UI/UX Premium de 2026:** Design baseado em Glassmorphism, com desfoque de fundo e animações espaciais.
* **Memória de Longo Prazo:** Histórico de mensagens salvo de forma segura com SQLite.
* **Avatares Dinâmicos e Gradientes:** Geração procedural de cores baseada no nome do usuário.
* **Sistema de Notificações:** Emblemas de mensagens não lidas, preview flutuante e sons de notificação.
* **Emoji Mart Integrado:** Experiência rica em emojis com auto-fechamento inteligente.
* **Indicador de Digitação:** Saiba exatamente quando a sua tripulação está respondendo.
* **Multitasking:** Minimize o chat sem perder a conexão ou as mensagens recebidas.

## 🛠️ Tecnologias Utilizadas
**Back-End:**
* Python 3
* Flask
* Flask-SocketIO (Comunicação Bidirecional)
* Flask-SQLAlchemy (Banco de Dados SQLite)
* Eventlet & Gunicorn (Servidor de Produção)

**Front-End:**
* HTML5 / CSS3 (Variáveis CSS, Flexbox, Animações Keyframe)
* JavaScript Vanilla (Lógica de estado, manipulação de DOM)
* Lucide Icons
* Emoji Mart API
